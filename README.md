## 📱 Controls

### Soul World 3D
- **WASD/Arrows**: Move
- **Mouse**: Look around
- **Click**: Lock pointer
- **🎯 Button**: Catch mode (hold & release)

### Other Games
- Touch/click interfaces
- Mobile friendly

## 🎯 Features

- ✅ 16 unique soul types (Profit, Love, Tax, Hybrid)
- ✅ 3D graphics with Three.js
- ✅ Mobile + Desktop support
- ✅ Save your collection (localStorage)
- ✅ Evolution system
- ✅ PLT scoring
